Aqui você vai ver a explicação de todo conteúdo sobre a tag <audio> </audio>, então vamos lá.
Existem diversas maneiras de se adicinonar um áudio no HTML, uma das maneiras mais simples é usando a tag abaixo.

    

        <audio src=""> </audio>
        
Aqui usamos o parametro autoplay para quando o navegador abrir a tela ela possa ser reproduzido, alguns ou outros navegadores poderá não reproduzir, vai variar muito de navegador, para nevegador.
Outro parametro aconselhavo para uso é o controls aonde você vai ter acesso as funções de áudio ( play, pause, volume, duração e até baixar o audio). 

<audio src="" controls> <audio>

        
    

    

        <audio preload="metadata" controls>
            <source src="midia/1 - Intro.mp3" type="audio/mpeg">
            <source src="midia/1 - Intro.ogg" type="audio/ogg">
            <source src="midia/1 - Intro.wav" type="audio/wav">
            
Infelizmente seu navegador não pode reproduzir esse audio Click aqui baixar esse audio.

Foi colocado varios<source src="" type="">para que o computador entenda que nem todos os sistemas operacionais são compativeis com o formato de música aquequadro, então foi colocado em três formatos MP3,OGG e WAV.
