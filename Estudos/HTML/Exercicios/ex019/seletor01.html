Comentário sobre o conteúdo dessa aula

Em HTML é id = "" em CSS é # ELE É UNICO
Uma regra do W3c é que não se pode usar id"nomeado" em outras tags.
Em HTML é class = "" em CSS é . ELE PODE SER PLURAL
Quando eu quero usar mais de um elemento para deixar ou o mesmo tamanho, ou mesmo modelo, agente usa uma class = "".

Do conteúdo acima colocamos style em vários h1's e h2's
