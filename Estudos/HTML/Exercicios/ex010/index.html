Usando link externos.

Primero vamos a explicação do conteúdo proposto.

Usar link's externo é adaptar um link de um site qualquer para a uma palavra descrita no seu texto ou sua frase.
Vamos ao um exemplo de código:

Você pode acessar o meu repositório publico no GitHub

para usar o link na frase a cima usei o parametro <a> </a> para melhor entender vou colar o código de digitação a baixo para melhorar usa compreenção.

        

    
            
  1. <p>Você pode acessar o meu repositório publico GitHub </p>
    2. 
            Apartir de "repositório publico no GitHub" dentro da tag <p> eu coloque o outra tag <a> </a>
            
  2. <p>Você pode acessar o meu <a>repositório publico GitHub</a> </p>
    3. 
            Além o atributo da tag <a> vem acompanhado de href="URL"
            
  3. <p>Você pode acessar o meu <a href="URL" target="_blank" rel="external">repositório publico GitHub</a> </p>
    4. 
            Pronto aqui esta a estrutura completa do link externo, porem existe mais duas funções dentro da tag <a> depois de href="", que são as target="_blank" e rel="external"
        

        
A target="_blank" -Ele vai abrir o link em uma nova aba.

Você também pode acessar o meu canal no youtube

Diferente do primeiro exemplo, este aqui se você experimentar clickar no link ele vai direcionar a página para outro destino sem deixar com que a pessoa passe a voltar a acessar sua página de inicio, se perdendo em vários outros conteúdos.

Links para Download

  • Baixar Conteúdo em PFD.
  • Baixar Conteúdo em ZIP

Aqui foi usado a <a> para criar o link de download, usei a mesma tag nos dois links.
Vamos ao Exemplo de Código:

        
Baixar Conteúdo em PDF
Baixar Conteúdo em ZIP

        <ul>
           1. <li>;Baixar Contéudo em PFD</li>
           1.2 <li><a herf="livro/Arquivo.pfd" download="Arquivo.pdf" type="application.pdf">Baixar Contéudo em PFD</a></li>
           2. <li>Baixar Contéudo em ZIP</li>
           2.2 <li><a herf="Arquivo.ZIP" download="Arquivo.ZIP" type="application.zip">Baixar Conteúdo em ZIP</a></li>
        
        </ul>
