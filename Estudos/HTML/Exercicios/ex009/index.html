Trabalhando com Listas

LISTAS ORDENADAS

Listas ordenadas são listas que seguem uma ordem, para manter uma padronização.

Para usarmos a função de listas ordenadas em nosso HTML ultilizamos a tag <ol> </ol> (Ordered List).
Eles são divididas em diferentes classes, tais elas são:
1 - Ordenação numerica;
a - Ordenação Alfabetica Minúsculo;
A - Ordenação Alfabetica Maiúsculo;
i - Ordenação de Numeros romanos Minúsculo;
I - Ordenação de Numeros romanos Maiúsculo.


Para delimitar a lista usamos a tag <li> </li> para você indicar o inicio da contagem você tem que usar o parametro Start=" "

Vamos ao exemplo a baixo vou printar o código e logo a baixo você verá o resultado!


    

        
            <ol type = "A" start= "4">
                <li>acodar</li>
                <li>tomar banho</li>
                <li>Tomar café</li>
                <li>escovar os dentes</li>
                <li>Trabalhar</li>
                <li>Almoçar</li>
                <li>Trabalhar</li>
                <li>Jantar</li>
                <li>Dormir</li>
            </ol>

Resultado do exemplo de código acima:
Porque começou a contar da letra 'D'?
Porque no inicio da tag <ol> eu coloquei o type="A" e iniciei a função start="4", então se na ordem alfabetica A é 1, D será 4.

  1. acodar
  2. tomar banho
  3. Tomar café
  4. escovar os dentes
  5. Trabalhar
  6. Almoçar
  7. Trabalhar
  8. Jantar
  9. Dormir

LISTAS NÃO ORDENADAS

Listas não ordenadas são listas que não precisa de etapas ou ordens para serem seguidas, mais que tem seus marcadores.

Para usarmos a funcão de lista não ordenadas, ultilizamos a tag <ul> </ul> (Unordered list) como usamos no exemplo anterior usaremos também a tag <li> </li> para delimitar a lista.

Essas Unordered list elas possuem um marcador padrão que é uma bolinha preta totalmente preenchida (Disc), mais que podem ser subtituida por outros marcadores como Disc, Circle e Square. Esse exemplo abaixo mostra o uso do marcador Disc:

Vamos ao exemplo código:

        

        <ul type= "dics">
            <li>Água</li>
            <li>Café</li>
            <li>Suco</li>
            <li>Chá</li>
            <li>Leite</li>
            <li>Iorgute</li>
            <li>Refrigerante</li>
        </ul>

Aqui essa a execução do código a cima:
Esse tipo de lista é mais usada para compras de supermecados, lojas, tabela de itens, menu de restaurantes etc.

  • Água
  • Café
  • Suco
  • Chá
  • Leite
  • Iorgute
  • Refrigerante

MESCLANDO O CONTEÚDO VISTO:

Podemos também criar lista mistas, listas de outras listas. Vejamos o exemplo a seguir:

    <ol>
        <li>Item 1</li>>
        <li>Item 2</li>>
            <ol type= "a">
                <li>Item 2.1</li>>
                <li>Item 2.2</li>>
                <li>Item 2.3</li>>
            </ol>>
        <li>Item 3</li>>
        <li>Item 4</li>>
        <li>Item 5</li>>
    </ol>>

Vejamos o resultado do exemplo feito:

  1. Item 1
  2. Item 2
    • Item 2.1
    • Item 2.2
    • Item 2.3
  3. Item 3
  4. Item 4

Notamos que no Item 2 exitem três casas (a,b,c) a baixo dela puxando para a direita, isso significa que elas são partes do Item 2,identificadas como filhas.

LISTA DE DEFINIÇÕES

É como se fosse um dicionário, temos o termo e suas descrições. É uma lista sem demarcações.
Toda alista de definições esta dentro da tag <dl> </dl> (Definition list). Cada termo é um <dt> (Definition term) e cada descrição é um <dd> (Definition Description).
Assim como os itens da lista, essas duas tags possuem fechamento opcional, segundo a referência oficial do HTML5.

Exemplos:

        <dl>
            <dt>HTML:</dt>
            <dd>Linguagem de criação e marcação de conteúdo de um site.</dd>
            <dt>CSS:</dt>
            <dd>Linguagem de marcação a criação do design de um site.</dd>
            <dt>JavaScript:</dt>
            <dd>Linguagem de programação de  interação do conteúdo de um site.</dd>
        </dl>

vejamos o resultado do exemplo feito a cima:

HTML:
Linguagem de criação e marcação de conteúdo de um site.
CSS:
Linguagem de marcação a criação do design de um site.
JavaScript:
Linguagem de programação de interação do conteúdo de um site.
