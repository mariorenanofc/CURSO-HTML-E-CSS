Tags do HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá ser ultilizada em cada caso.

  • Parágrafo
  • Título
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Lista Numericas ou Lista Ordenadas
  • Lista Não Ordenadas ou demarcadores
